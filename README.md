# Khalil Asaad Mustafa Portfolio

A simple Next.js portfolio with a polished retro terminal style. The site is static, responsive, and ready to deploy to Vercel.

## Run Locally

```bash
npm install
npm run dev
```

Open `http://localhost:3000`.

## Edit Portfolio Content

All visible portfolio content lives in:

```text
data/portfolio.ts
```

Update that file to edit profile text, links, projects, skills, experience, education, certifications, and contact details. Project GitHub and demo buttons are currently marked as `TODO`; replace those placeholder URLs when the real links are available.

## Deploy To Vercel

1. Push the project to a GitHub repository.
2. Import the repository in Vercel.
3. Keep the default Next.js settings.
4. Deploy.

Useful commands:

```bash
npm run build
npm run start
```
